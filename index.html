Introducción

Logical Framework es una herramienta profesional para recolectar datos y generar análisis con la metodología del marco lógico y el árbol de problemas. Permite registrar la información de la empresa, capturar problemas/causas/efectos y producir borradores de análisis e informes.

Objetivo

Proveer una interfaz sencilla y práctica para que empresarios y analistas registren el contexto, identifiquen el problema central, sus causas y efectos, y generen un marco lógico inicial y reportes exportables.

Uso — explicación completa

  1. Empresa: Complete los datos básicos (nombre, sector, ubicación, tamaño y descripción) y presione "Guardar".
  2. Recolección: Agregue elementos seleccionando el tipo (problema / causa / efecto). Puede editar, eliminar y vincular elementos entre sí (relaciones simples).
  3. Análisis: Genere un borrador del marco lógico o visualice un árbol simplificado a partir de los registros.
  4. Informes: Vea resúmenes y listas; exporte/importe datos JSON desde el menú.

Ejemplo

1) En "Recolección" agregue un elemento tipo problema titulado "Baja productividad".

2) Agregue causas como "Falta de capacitación" y efectos como "Pérdida de clientes".

3) Vincule la causa con el problema y genere el borrador del marco lógico en "Análisis".
